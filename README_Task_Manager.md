# Task Manager

## Project Overview

This project is a simple console-based Task Manager built with Python.
It allows users to add, view, search, and mark tasks as completed while
practicing core Python programming concepts.

------------------------------------------------------------------------

## Objective

Practice the following concepts:

-   Functions
-   Lists
-   Dictionaries
-   Loops
-   Conditional Statements
-   CRUD Operations

------------------------------------------------------------------------

## Technologies

-   Python 3
-   Jupyter Notebook / Google Colab

### Libraries Used

-   No external libraries are required.

------------------------------------------------------------------------

## Features

-   Add new tasks
-   View all tasks
-   Search tasks by ID
-   Mark tasks as completed
-   Display task status
-   Manage tasks using lists and dictionaries

------------------------------------------------------------------------

## How to Run

1.  Open the notebook in Jupyter Notebook or Google Colab.
2.  Run all cells.
3.  Follow the on-screen prompts to manage tasks.

------------------------------------------------------------------------

## Concepts Covered

-   Functions
-   Lists
-   Dictionaries
-   Loops
-   Conditional Statements
-   CRUD Operations

------------------------------------------------------------------------

## Learning Outcomes

After completing this project, students will be able to:

-   Create reusable functions.
-   Store and manage data using lists and dictionaries.
-   Perform Create, Read, Update, and basic task management operations.
-   Build a simple command-line application.
